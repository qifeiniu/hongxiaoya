## 0. 根节点 
- 相亲交友APP 
- 相亲交友APP功能需求思维导图3.0 

## 1. 登录注册与信息完善 
- 支持微信登录：微信SDK 
- 短信验证码登录：阿里云短信 
- 未登录仅允许浏览app，不可使用app任何功能 
- 必须完成注册并登录方可使用APP 
- 新用户注册即赠送100鸭蛋 
- 完善个人信息：头像、昵称、性别、年龄、认证等；填写个人介绍优先展示推荐 
- 实名认证：上传身份证件（对接API接口） 
  - OCR识别 + 活体检测（必要功能：人脸识别和实名验证） 
- 学历认证：学历证件资料认证端口（后台人工审核） 
- 实名注册会员均可：聊天、送礼物、报名相亲活动、发布动态、使用漂流瓶等 

## 2. 底部导航栏五大模块 

### 2.1 首页 
- 推荐机制与展示 
  - 实时推荐在线异性用户（按活跃度/匹配度排序） 
  - 大图展示效果，突出视觉吸引力 
  - 按上线时间优先推荐实时在线人员，其次活跃度、匹配度 
  - 显示几秒前/几分钟前在线、最近在线时间、二人之间距离 
  - 首页推荐仅展示异性 
- 右上角功能 
  - 分享转发：可转发给微信好友或微信群 
  - 筛选按键（仅会员支持使用） 
    - 居住地、年龄、身高、学历、收入、籍贯、户口所在地、婚姻（未婚/离异） 
    - 说明：筛选仅充值会员可有权限设置 
- 首页展示信息（个人卡片/主页信息） 
  - 个人照片：可多张滑动 
  - 认证状态：展示身份证号码前3位 
  - 学历认证：XX学校、专科/本科/硕士/博士、年收入 
  - 昵称、头像、是否在线、是否认证、距离、地区、年龄星座、身高、体重、学历、学校、职业、籍贯、户籍地 
  - 模块字段：基础资料 / 家庭背景 / 兴趣爱好 / 爱情宣言 / 我的理想另一半 / 期待什么样的生活 
  - 朋友圈个人动态：首页仅展示第一条；入口框点击进入会员发布的朋友圈主页 
  - 个人介绍区域：同步展示个人照片 
  - 我的标签：用户自己编辑（例：性格开朗、独立、积极生活） 
  - 我的认证 
    - 头像认证：完成人脸比对 
    - 实名认证：居民身份证 
    - 学历认证：提交毕业证/毕业证编号 
- 首页右侧按键交互 
  - 玫瑰花 
    - 系统扣20个鸭蛋（钱包可查明细） 
    - 首页自动切换下一位会员主页 
    - 消息页自动开启对话框：显示“赠送玫瑰花” 
    - 赠送后可免费再发送一条信息；对方回复则双方自动开启聊天模式 
  - 红心（喜欢） 
    - 归入喜欢列表并切下一位会员主页，不扣鸭蛋 
    - 双方互相喜欢则自动匹配开启聊天模式 
  - 谁喜欢了我 
    - 入口查看“喜欢了我的列表” 
    - 付费查看：20鸭蛋/人 
  - 不喜欢 
    - 切换下一位会员主页，归入不喜欢列表，不扣鸭蛋 
  - 情景规则 
    - A送B玫瑰花，B点A喜欢 → 自动进入聊天（可直接回复/聊天） 
    - A点B喜欢，B给A送玫瑰花 → 自动进入对话聊天模式 
- 喜欢/不喜欢列表 
  - 我喜欢的用户 
  - 我不喜欢的用户 
- 解锁微信号（放基础资料板块） 
  - 会员：50鸭蛋解锁 
  - 非会员：100鸭蛋解锁 
- 举报/屏蔽：按键放在页面最下边 
- “鸭蛋”互动机制（与聊天/礼物/视频相关） 
  - 每日签到：普通用户送10个鸭蛋；会员签到鸭蛋翻倍（后台设定） 
  - 赠送礼物后，若对方回复信息 → 双方立即开通1对1文字聊天权限 
  - 若对方未回复 → 赠送方需等待对方回复后才可开启聊天 
  - 聊天框可重复赠送礼物，次数不限 
  - 虚拟礼物（消耗鸭蛋）：玫瑰花20 / 跑车50 / 火箭100 
  - 1对1视频聊天：按分钟计费 
    - 2分钟/60鸭蛋；4分钟扣120鸭蛋；先扣后聊；超时自动扣 
    - 仅扣视频发起方鸭蛋，接收方不扣 

### 2.2 发现 
- 上半屏：智能推荐在线用户卡片流 
  - 展示：头像、昵称、性别、年龄、距离、在线状态、1–3个兴趣标签 
  - 仅展示异性，优先同城用户 
  - 可展示实时在线人数/可设置在线人数 
  - 点击进入对方个人主页查看完整资料 
- 下半屏：UGC信息流（图文/视频/文字动态） 
  - 优先显示同城信息 
  - 发布支持：图片（多图）/短视频（≤60秒）/纯文字（≤500字） 
  - 内容合规：自动检测、敏感词过滤、百度内容过滤SDK（违规可封禁，时间后台设置） 
  - 互动：点赞、评论、转发（转发至好友或微信等外部平台） 
  - 话题：会员可自由创造话题，其他会员参与（例 #2026我要脱单） 
  - 发布无权限限制（所有用户均可发布） 
  - 发帖展示：头像、昵称、年龄、坐标、户籍地、学历、身高 
  - 信息流可下滑浏览其他动态 

### 2.3 相亲 
- 活动列表：展示当前及即将开始的线上相亲活动 
- 活动创建者：具备“红娘”身份认证的用户创建与管理 
  - 基础条件：需给平台拉不低于10个人注册会员才可申请 
- 红娘等级与分成 
  - 拉10人：白银红娘，分成30% 
  - 拉50人：黄金红娘，分成30%（头像标签变更） 
  - 拉200人：白金红娘，分成32%（头像标签变更） 
  - 拉300人：钻石红娘，分成35%（头像标签变更） 
- 红娘创建在线相亲需设置参数 
  - 活动名称、封面图、简介、活动要求说明（红娘编辑发布） 
  - 性别配额：男X名、女Y名；报名总限额100人 
  - 报名费用：仅微信/支付宝；不低于10元/人；红娘可自由设置 
  - 活动时间：开始时间、持续时长、是否循环举办 
  - 报名截止时间、审核机制：自动通过/红娘人工审核 
  - 退款规则：无法正常举办可申请/自动退款；若正常举办则不支持退款 
- 报名后流程 
  - 系统提醒及活动倒计时 
  - 到时间进入专属多人视频房间 
  - 房间能力：9宫格视频、文字公聊、举手发言（可选） 
  - 发言者可大窗口展示；未发言仅小头像 
  - 红娘房主权限：禁言、移出房间、控麦、置顶公告、结束活动等 
- 活动页报名用户列表与隐私 
  - 可看已报名用户列表 
  - 不可点头像看主页 
  - 查看会员主页：20鸭蛋/人 
  - 仅展示报名用户小头像，不展示其他隐私信息 
- 活动内聊天规则 
  - 参加在线相亲的会员可自由选择喜欢对象聊天 
  - 自动可开启聊天对话框，不需要送花开启 
- 分账提现（重要） 
  - 需后台设置自动分账系统 
  - 活动结束一定时间内红娘可自动将佣金提现到银行卡 

### 2.4 消息 
- 消息列表页：按时间倒序 + 未读标红 + 显示未读条数 
- 筛选：已读/未读、时间 
- 分类Tab：私聊、系统通知、活动提醒、喜欢/礼物通知 
- 私聊会话：支持置顶（长按→置顶/解除），或可选择解除聊天权限 
- 系统通知类型 
  - 有人点击喜欢（小心心） 
  - 有人赠送礼物 
  - 相亲活动：报名成功/审核通过/即将开始 
  - 漂流瓶：被捞起/回复 
  - 认证进度提醒、签到奖励发放 
- 私聊界面功能 
  - 文字输入、语音、打招呼表情包、图片/视频发送（≤10MB） 
  - 送礼物、视频通话（60鸭蛋/2分钟）、语音通话（60鸭蛋/2分钟） 
  - 消息3分钟内长按撤回 
  - 点击头像进入主页 
  - 语音/视频通话：需双方同意并付费；发起方付费，先扣后通话，接听免费 
  - 礼物：玫瑰20 / 跑车50（带特效）/ 火箭100（带特效） 
- 搜索：输入ID可搜会员和聊天记录；昵称搜会员等 

### 2.5 我的 
- 个人中心总览 
  - 头像（可上传）、昵称、认证标识（已实名/学历认证/头像认证）、ID号 
- 个人信息快捷入口 
  - 编辑资料、填写个人微信号（重要） 
  - 可上传多张个人照片 
  - 隐私设置、账号安全 
  - 个人主页编辑完预览 
  - 展示字段：学历（XX学校、专科/本科/硕士/博士）、年收入、地区、年龄、星座、身高、体重、职业、籍贯、户籍地、身份证号等 
- 核心功能模块 
  - 账户资料：绑定手机号短信登录、微信登录 
  - 浏览访客（仅会员可见）：最近三个月访问我主页用户（头像+昵称+访问时间） 
  - 个人动态管理：发布/编辑/删除；可见范围（公开/好友/仅自己） 
  - 我的认证：对接公安部门三方身份证实名认证（OCR+活体检测）（必要功能） 
  - 使用帮助：操作指引视频、新手引导 
  - 联系客服：在线客服（工作时间）、留言反馈、投诉建议 
  - 办理会员：开通路径、权益对比表、续费提醒 
  - 漂流瓶：入口按钮进入投放/捞取页 
  - 签到中心：非会员10鸭蛋/天，会员20鸭蛋/天；签到赠送鸭蛋有效期1个月 
  - 充值中心（微信/支付宝） 
    - 28元200 / 68元600 / 128元1300 / 198元2200 / 298元3500 / 398元5000 
    - 鸭蛋期限：充值购买无期限；非充值获取（签到/拉人）每月1号清零 
  - 我的钱包：红娘钱包有提现功能 
  - 礼物商城：鸭蛋兑换礼物（玫瑰20、跑车50、火箭100）用于对话框礼物选项 
  - 红娘入驻：申请入口、资质要求、审核流程；完成邀请满10个会员注册 
  - 邀请会员注册：每邀请1人注册并完成认证奖励100鸭蛋；生成APP邀请二维码；个人ID号码 
- 会员权益体系（核心特权） 
  - 价格：1个月28 / 3个月68 / 6个月98 / 12个月128 
  - 喜欢：每日不限次数 
  - 浏览访客：查看最近3个月访客（含头像昵称访问时间） 
    - 非会员解锁：20鸭蛋/人 
  - 看谁喜欢我：非会员解锁20鸭蛋/人 
  - 漂流瓶：会员每日捡/扔各20；非会员各5 
  - 私聊已读回执：显示“对方已读” 
  - 签到：会员20/天，非会员10/天 
  - 视频通话免排队：优先接入视频服务器降低延迟 
  - 优先推荐给喜欢的人 
  - 会员筛选功能 
  - 漂流瓶功能说明 
    - 非充值会员：捡5/丢5；充值会员：捡20/丢20 
    - 优先同城/同省捡到；仅可被异性捡到 
    - 收到漂流瓶后回复：点击发送自动生成新漂流瓶，不占每日名额 
    - 双方自动私聊后：聊天渠道有效期1个月；到期自动关闭；查看对方主页扣20鸭蛋 
    - 内容限制：文字200字内、图片限1张 
    - 超出数量：捡/丢漂流瓶 10鸭蛋/次 

## 3. 鸭蛋经济体系（汇总模块） 
- 获取方式 
  - 每日签到：普通10，会员20 
  - 新用户注册赠送100 
  - 充值购买（多面额套餐，含限时赠蛋活动） 
  - 邀请用户注册：100鸭蛋/人 
- 消耗场景 
  - 对话框礼物：玫瑰20 / 跑车50 / 火箭100 
  - 视频聊天：2分钟=60鸭蛋（按分钟计费） 
  - 捞取漂流瓶：10鸭蛋/次 
  - 非会员查看访客：20鸭蛋/人 
  - 非会员查看谁喜欢我：20鸭蛋/人 
  - 查看会员微信号：50鸭蛋/次 
  - 首页送玫瑰花开启对话框：20鸭蛋/次
